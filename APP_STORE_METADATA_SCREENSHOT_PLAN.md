## Description (draft)
Card Counting Protocol 21 is an educational training app for blackjack advantage play.

Build real card-counting discipline through guided modules:
- Basic strategy
- Running count accuracy
- True count conversion
- Deviation recognition
- Decision quality analytics

Use Decision Lab sessions to practice high-pressure scenarios and track:
- Edge trajectory
- Error cost
- Risk-of-ruin sensitivity
- Progress over time

Premium unlocks deeper analytics, advanced drills, and expanded training modules.

This app is an educational training tool and does not provide real-money gambling or simulated wagering gameplay.
